考场安排系统 使用说明
========================================

一、软件简介
《考场安排工具》是专为第二实验中学教务场景设计的全自动考场编排工具，无需代码基础、无需复杂操作，只需导入学生成绩数据，即可一键完成S型均衡分班、准考证号生成、考场座位分配、收尾考场容量自动计算，大幅减轻考试编排的人工工作量。

二、核心功能亮点
- 智能排序：自动识别「总分/年级排名」列，优先按总分降序、无总分则按年级排名升序排序
- 均衡分班：S型交叉分配学生，保证各考场成绩分布均衡，避免同班级学生集中
-自动算容：仅需设置前N个考场容量，最后一个收尾考场自动计算，无需手动调整
- 模板化操作：自带标准Excel模板，直接填写数据即可，无需修改格式
- 即开即用：单EXE文件，无需安装Python、无需配置环境，Windows电脑双击直接运行
- 路径固定：自动在桌面生成规范文件夹，输入模板、输出结果分类存放，不会出现文件找不到的问题

三、运行环境
- 操作系统：Windows 7 / Windows 10 / Windows 11（32/64位全兼容）
- 运行要求：无需安装任何额外软件，双击EXE即可启动
- 权限说明：建议右键选择「以管理员身份运行」，避免文件写入权限不足

四、快速上手步骤
========================================
第一步：启动软件
双击「考场安排系统.exe」启动程序，软件会自动在桌面创建「考场打印材料」文件夹，内含「输入文件」「输出结果」两个子文件夹，所有模板和生成结果均存放于此。

第二步：准备学生成绩数据
1. 打开桌面路径：考场打印材料/输入文件/学生原始成绩.xlsx
2. 按照模板格式填写学生数据：
   ▶ 必填列：姓名、班级（班级必须填写纯数字，例：1、2，不可填「初一1班」）
   ▶ 选填列：总分、年级排名（二选一填写即可，系统自动识别排序依据）
   ▶ 注意：请勿修改模板的列名，否则系统无法识别数据
3. 填写完成后，关闭Excel文件，避免文件被占用导致生成失败。

第三步：设置考场容量（2种方式任选）
▶ 方式1：软件内手动设置
- 软件启动后默认添加9个考场，单考场默认容量50人
- 点击「添加考场」可新增考场，点击考场右侧「删除」可移除考场
- 直接在输入框中修改每个考场的容量（必须为正整数）

▶ 方式2：模板导入（推荐，适合固定考场场景）
1. 打开桌面路径：考场打印材料/输入文件/考场容量模板.xlsx
2. 填写前N个考场的固定容量，最后一个收尾考场无需填写，系统会自动计算
3. 回到软件，点击「从模板导入」，即可一键导入所有考场容量
4. 也可点击「导出当前为模板」，把软件内设置的容量保存为模板文件

第四步：设置基础信息
- 考号前缀：可自定义准考证号前缀，默认701020，最终考号格式为「前缀+3位流水号」（例：701020001）
- 排序规则：系统自动识别，无需手动操作，优先按总分降序，无总分则按年级排名升序

第五步：一键生成考场安排
点击软件底部的「开始生成安排」按钮，等待弹窗提示「生成成功」，即完成所有编排工作。

第六步：查看最终结果
生成的文件固定存放路径，软件界面也标注了完整路径：
1. 学生考试信息.xlsx → 桌面「考场打印材料/输入文件」文件夹
   内含：准考证号、姓名、班级、考场号、座号，可直接用于打印学生准考证、考场座位表
2. 考场容量.xlsx → 桌面「考场打印材料/输出结果」文件夹
   内含：考场号、设置容量、实际人数、起始考号、截止考号，可直接用于监考安排、考场物料准备

五、文件夹&文件说明
========================================
桌面/考场打印材料/
 ├─ 输入文件/
│  ├─ 学生原始成绩.xlsx      # 学生成绩模板，填写后用于生成安排
│  ├─ 考场容量模板.xlsx      # 考场容量模板，可导入软件批量设置
│  └─ 学生考试信息.xlsx      # 生成的最终学生考场信息表
└─ 输出结果/
   └─ 考场容量.xlsx           # 生成的最终考场容量统计表

六、功能模块说明
========================================
1. 文件设置区域
- 成绩文件：显示当前使用的学生成绩文件路径，可点击「浏览」选择其他成绩文件
- 容量模板：显示当前使用的考场容量模板路径，可点击「浏览」选择其他模板文件
- 学生信息输出：只读显示学生考试信息文件的完整保存路径
- 考场容量输出：只读显示考场容量统计表的完整保存路径

2. 基本信息区域
- 考号前缀：自定义准考证号前缀，支持数字/字母组合
- 排序规则：显示系统的排序逻辑，无需手动操作

3. 考场容量设置区域
- 添加考场：新增一个考场输入行，自动编号
- 从模板导入：从当前设置的容量模板文件中导入考场容量
- 导出当前为模板：把软件内设置的考场容量，导出到当前设置的模板路径
- 滚动区域：显示所有手动设置的考场，支持修改容量、删除考场

4. 底部功能按钮
- 重置成绩模板：恢复「学生原始成绩.xlsx」为初始模板，清空所有自定义数据
- 开始生成安排：执行核心编排逻辑，生成最终结果文件
- 退出：关闭软件

七、常见问题FAQ
========================================
Q1：双击软件没反应/一闪而过怎么办？
A1：
1. 右键软件，选择「以管理员身份运行」
2. 检查软件所在路径是否有中文/空格/特殊符号，建议放在D盘根目录的纯英文文件夹中
3. 关闭杀毒软件/Windows Defender实时防护，或把软件加入白名单（属于打包正常误报，软件无任何病毒代码）

Q2：点击生成，提示「成绩文件不存在」怎么办？
A2：检查「成绩文件」路径是否正确，或点击「重置成绩模板」重新生成模板文件，再填写数据。

Q3：提示「前N个考场总容量超过学生总数」怎么办？
A3：减少手动设置的考场数量，或降低单考场容量，保证手动设置的总容量小于学生总数，系统会自动生成最后一个收尾考场。

Q4：生成的文件找不到怎么办？
A4：所有文件都固定存放在桌面的「考场打印材料」文件夹中，学生信息在「输入文件」子文件夹，考场统计在「输出结果」子文件夹，软件内也标注了完整的文件路径。

Q5：Excel打开提示「文件被占用」怎么办？
A5：关闭所有打开的Excel窗口，再重新点击生成按钮，避免Excel文件被锁定导致无法写入。

Q6：班级列填了文字，生成报错怎么办？
A6：班级列必须填写纯数字（例：1、2、3，不能填「初一1班」「七年级2班」），否则系统无法进行S型分班排序。

Q7：杀毒软件删除了exe文件怎么办？
A7：这是PyInstaller打包的Windows系统正常误报，解决方法：
1. 在杀毒软件的「隔离区」中恢复exe文件
2. 把软件所在文件夹加入杀毒软件的信任区/白名单
3. 生成时暂时关闭实时防护，生成完成后再开启
